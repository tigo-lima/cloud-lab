# Azure Cloud Engineer Lab

This project is a hands-on cloud engineering lab focused on Azure.

## Objectives
- Build infrastructure using Terraform
- Deploy applications using Docker
- Automate deployments with CI/CD
- Implement monitoring and security
- Evolve to Kubernetes (AKS)

## Tech Stack
- Azure
- Terraform
- Docker
- GitHub Actions

## Progress

### Week 1
- Local environment prepared
- Azure CLI configured
- Terraform installed
- Docker installed
- SSH key created
- VS Code workspace prepared

### Week 2
- Azure infrastructure created manually
- VNet, subnet, NSG, public IP and NIC created
- Ubuntu VM deployed
- SSH access validated
- Nginx installed and running
- HTTP service exposed publicly

### Week 3
- Infrastructure recreated with Terraform
- Separate Azure resource group deployed with IaC
- NSG, VNet, subnet, NIC, public IP and Linux VM created automatically
- Terraform apply completed successfully
- Public IP output generated automatically
